Variables in JavaScript

Variables are containers for storing values.

And we can give that name in three ways.

  • Using let keyword
  • Using const keyword
  • Using var keyword

NOTE

  • Always use const if the value should not be changed.
  • Always use const if the type should not be changes (Arrays, Functions, Objects and RegExp).
  • Only use let if you can't use const.
  • You cannot re-declare a variable declare with let or const.
  • Variables declare with let and const have Block Scope.
  • Variables declare with let and const cannot be re-declared in the same Scope.
  • Variables declare with var always have Global Scope.
  • let & const does not bind to this.
  • let & const are not hoisted.
  • const Variables must be assigned a value when they declared.

Constant Objects and Arrays

The keyword const is little misleading

It does not define a constant value. It defines a constant reference to a value.

Because of this you can NOT:

  • Reassign a constant value
  • Reassign a constant array
  • Reassign a constant object

But you CAN:

  • Change the elements of constant array
  • Change the properties of constant object
