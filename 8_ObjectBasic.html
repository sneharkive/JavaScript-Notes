Object Basic

Objects are containers for Properties and Methods.

Properties are named Values. Methods are Functions stored as Properties.

Properties can be primitive values, functions, or even other objects.

JavaScript Objects are Mutable. Since, they are addressed by reference, not by value.

Define a JavaScript Object in 3 ways

  • Using an Object Literal(also called object initializers)
  • Using the new Keyword
  • Using an Object Constructor

    • JavaScript Object Methods

    Methods are actions that can be performed on objects. Methods are function definitions stored as property values.
